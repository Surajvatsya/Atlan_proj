# How to decrease Page Load Time?
1.Minify Js and CSS file
<br>
2. Asynchronous Loading of Js and CSS file <br>
3. Defer Large Size Elements <br>
4. Media comressions using plugins <br>
5. Enable browser caching<br>
6. Apply CDN<br>
